Detailed Description

This header file defines the Inet::TCPEndPoint class, where the CHIP Inet Layer encapsulates methods for interacting with TCP transport endpoints (SOCK_DGRAM sockets on Linux and BSD-derived systems) or LwIP TCP protocol control blocks, as the system is configured accordingly.
