Detailed Description

template<class ImplClass>
class chip::DeviceLayer::Internal::GenericThreadStackManagerImpl_FreeRTOS< ImplClass >

Provides a generic implementation of ThreadStackManager features that works on FreeRTOS platforms.

This template contains implementations of select features from the ThreadStackManager abstract interface that are suitable for use on FreeRTOS-based platforms. It is intended to be inherited, directly or indirectly, by the ThreadStackManagerImpl class, which also appears as the template's ImplClass parameter.
