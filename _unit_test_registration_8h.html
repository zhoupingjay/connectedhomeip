Registers a unit test suite runner.

Adds a function of the signature int(*)() to the list of test runners which should be invoked when chip::RunRegisteredUnitTests() is called.

Example:

int MyTestSuiteRunner()
{
if (!RunSomeTests())
return -1; // test failure
return 0; // test success
}
