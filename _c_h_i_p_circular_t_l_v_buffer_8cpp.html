Detailed Description

This file implements the circular buffer for TLV elements. When used as the backing store for the TLVReader and TLVWriter, those classes will work with the wraparound of data within the buffer. Additionally, the TLVWriter will be able to continually add top-level TLV elements by evicting pre-existing elements.
