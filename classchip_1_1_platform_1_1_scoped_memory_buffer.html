Detailed Description

template<typename T, class MemoryManagement = Impl::PlatformMemoryManagement>
class chip::Platform::ScopedMemoryBuffer< T, MemoryManagement >

Represents a memory buffer allocated using chip::Platform::Memory*Alloc methods.

Use for RAII to auto-free after use.
