Detailed Description

Defines a data structure that can store a set of uint64_t values in not contiguous indexes.

The set is initialized with a constant capacity. The data structure supports serialize/deserialize operations, where, serialize converts the set to a base64 string, and deserializing the base64 string reconstructs the original set.

The data is stored such that serialized data can be deserialized correctly on different machine architectures.
