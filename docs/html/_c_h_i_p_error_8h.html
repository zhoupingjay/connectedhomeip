Detailed Description

This file defines error constants for the CHIP core subsystem.

Error types, ranges, and mappings overrides may be made by defining the appropriate CHIP_CONFIG_* or CHIP_CONFIG* macros.

NOTE WELL: On some platforms, this header is included by C-language programs.

Macro Definition Documentation

#define _CHIP_ERROR (   e)    _CHIP_CONFIG_ERROR(e)
